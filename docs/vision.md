# Life OS — Product Vision

## Миссия

Life OS — персональная операционная система, которая помогает осознанно управлять жизнью, достигать долгосрочных целей и снижать количество ежедневных решений за счет системы, а не силы воли.

## Основные принципы

1. Система важнее мотивации.
2. Результат важнее активности.
3. Минимум ручного ввода.
4. Один источник истины для каждой сущности.
5. Life OS советует, но не контролирует.
6. Все области жизни рассматриваются как единая система.
7. Центр операционного управления — текущий день.
