# Life OS — Domain Model v0.1

## Основная цепочка

Identity → Domain → Vision → Goal → Project → Outcome → Action → Day

## Поддерживающие сущности

- Event
- Habit
- Metric
- Review
- Decision
- Resource

## Центральная сущность

`Day` объединяет долгосрочные приоритеты, доступные ресурсы, календарные события, привычки, показатели и выбранные действия.
